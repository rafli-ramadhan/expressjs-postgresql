# Express + Sequelize + MySQL with @Hapi/Joi (CRUD and Auth Application)

## Requirement

1. Node.js
2. MySQL

## Set Up :rocket:

```
npm install
```

```
npm run dev
```

App will be running on port 5000 (http://localhost:5000).

## API Endpoint for Development 

http://127.0.0.1:5000
